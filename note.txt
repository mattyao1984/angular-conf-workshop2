================= Init Project with SSR ======================
1. create project with Angular CLI:
ng new [NAME] --routing --prefix [PREFIX]

2. Enhance the project to be a PWA:
ng add @angular/pwa

3. Enable SSR:
ng g universal --client-project piggybank

4. Build them both:
ng build --prod && ng run piggybank:server

5. Enter the build project and serve it using http-server
npm i -g http-server
cd dist/piggybank
http-server

================= Add Firebase ======================
1. Install firebase tools and CLI
npm i -g firebase-tools

2. firebase login
3. firebase console (from online dashboard)
- Enable firestore under the database section

4. firebase init
5. firebase use --add
6. firebase deploy
